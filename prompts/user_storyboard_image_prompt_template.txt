Frame / shot policy:
- Use the requested shot count. If no count is given, choose the smallest number of shots that clearly tells the story.
- Make reading order unmistakable: left-to-right, top-to-bottom.
- Use only tiny labels such as S1, S2, S3. Do not add dense captions, speech bubbles, fake UI screens, or long readable text.
- The image must remain understandable even if every label is ignored.

Continuity policy:
- Assign stable IDs to recurring elements before describing shots:
  - C1, C2 = recurring characters.
  - V1 = recurring vehicle or machine.
  - P1, P2 = important props.
  - L1, L2 = recurring locations.
  - D1 = important device/screen if needed.
- Keep character appearance, vehicle shape, props, lighting, and geography consistent across all frames.
- Use tail-frame continuity: every shot should feel like the next beat after the previous shot, not a disconnected poster.
- Clarify what changes from shot to shot: camera distance, character action, object state, emotional tension, and reveal.

Director / camera policy:
- Include a varied sequence of shot types, chosen for storytelling:
  - establishing shot
  - interior medium shot
  - close-up insert
  - POV shot
  - over-the-shoulder or foreground/background blocking
  - exterior wide shot
  - reaction shot
  - lingering aftermath shot
  - drone or top-down shot if useful
- Show action visually through posture, movement, framing, light, and object placement.
- Use sound effects only as visual hints: small sound-wave icon, subtle notification symbol, or tiny popup if necessary.
- Do not rely on written explanations inside the artwork.

Visual style policy:
- Follow the user’s requested style first.
- Prefer clear silhouettes, readable staging, strong value contrast, and restrained color scripting.
- Avoid real brand logos, readable license plates, celebrity likenesses, sensational violence, sexualization, or unrelated background clutter.
- If text is essential, keep it minimal and generic, such as “₩5,000” or “arrival,” not full UI paragraphs.

Output format:
- One complete storyboard-reference image only.
- The image should look like a director’s planning board for a short video.
- It should communicate sequence, blocking, camera grammar, repeated props, and final emotional beat clearly.
